## 存储模型

```python
# 保存整个网络
torch.save(model, 'model.pkl') 
# 保存网络中的参数, 速度快，占空间少
torch.save(model.state_dict(), 'model_parameter.pkl')
#--------------------------------------------------
#针对上面一般的保存方法，加载的方法分别是：
model_dict=torch.load('model.pkl')
model_dict=model.load_state_dict(torch.load('model_parameter.pkl'))
```
